## Task entry format

```markdown
- [ ] **M5.1** — short imperative title
  - **Goal:** what shipping looks like
  - **Files:** primary files touched
  - **Depends on:** M4.15, M5.0
```

Mark complete with `[x]` and a date. Never renumber tasks once they're committed.
